# Task Manager

## Description
Task Manager is a command-line application designed to help users manage
their tasks efficiently. It provides functionality for registering users,
adding tasks, viewing all tasks, and viewing tasks assigned to a specific
user. The project aims to simplify task management and increase
productivity by providing a convenient interface for organizing and
tracking tasks.

## Installation
To install and run the Task Manager application locally, follow these
steps:

1. Clone the project repository from GitHub: [link-to-repo]
2. Ensure you have Python 3.x installed on your machine.
3. Install the required dependencies by running the following command:
    pip install - r requirements.txt

4. Set up the necessary data files:
- Create a 'users.txt' file to store user credentials.
- Create a 'tasks.txt' file to store task details.

## User Guide
After installing the Task Manager, follow these instructions to use the
application:

1. Open a terminal or command prompt.
2. Navigate to the project directory.
3. Run the following command to start the Task Manager:
    task_manager.Python

4. The application will display a menu with several options:
- 'r': Register a new user.
- 'a': Add a new task.
- 'va': View all tasks.
- 'vm': View tasks assigned to you.
- 'gr': Generate reports.
- 'ds': Display statistics.
- 'e': Exit the program.

5. Select an option by entering the corresponding letter or command.
6. Follow the prompts and instructions provided by the application to
perform various tasks, view task information, generate reports, or
display statistics.
